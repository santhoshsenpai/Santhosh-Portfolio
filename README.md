# 🚀 Personal Portfolio Website

Welcome to my personal portfolio repository! This website is built entirely from scratch to showcase my skills, projects, and custom design implementations. By managing my own styling and asset architecture, I have kept the layout completely independent of heavy framework overhead.

## 🌟 Key Features

- **Custom-Crafted Layout:** Designed using HTML5 and CSS3, leveraging native modern features like Flexbox for fully aligned components.
- **Persistent Navigation:** Features a fixed, sleek global navigation bar that remains anchored to the top of the viewport during active scrolling.
- **Seamless Page Transitions:** Uses native CSS smooth-scrolling configurations for elegant anchor jumping across sections.
- **Custom-Engineered Contact System:** Used clourflare icons with customized CSS.
- **Direct Asset Downloads:** Includes fully configured anchor hooks enabling instant downloads of my professional resume in crisp PDF format.

---

## 🛠️ Built With

- **HTML5:** Semantic architecture for structured and accessible content.
- **CSS3:** Custom stylesheet utilizing Google Fonts ('Poppins') and modular UI styling configurations, built with dark & light blue with white theme.
